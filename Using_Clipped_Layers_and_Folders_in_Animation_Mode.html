Using Clipped Layers and Folders in Animation Mode

This page goes over how layers, clipping masks, and folders work when using FireAlpaca in animation mode.

FireAlpaca can be used to create animated gifs. In animation mode, layers function as frames, with the bottom layer set as frame one and so on upwards in the "Layer" window. For simple animation, one-layer frames should be sufficient. This page goes over how to incorporate clipping masks and folders to create more complex animations.

When folders are added, each folder functions as a frame in the same way as a single layer.

Using folders can be helpful when you want parts of an illustration to be animated separately. For example, if you are animating a character standing in windy weather, you may want to move only the hair and leave the rest of the character static. If using one layer, it can be difficult to keep static parts the same when erasing the parts to be animated. Additionally, if separated parts meant to appear together are not placed in a folder, each part will appear as a separate frame.

Instead, you can draw separately animated parts on different layers, then place them into the same folder so that they still appear as one frame. You can then duplicate the folder as necessary with the separated folders in place, which allows you to then adjust each part individually per frame.
Folders can also be useful if you want to animate using illustrations with separate line art and color. Simply place your line art and color layers into the same folder and they will appear together as the same frame.

Using folders also allows you to use the clipping mask. If layers are not placed together in a folder in animation mode, they will appear as separate frames in the animation; a clipped layer will thus not appear on the layer it is clipped to, but as its own, separate frame. Use a folder to group layers that are clipped together so that they will appear as intended.
