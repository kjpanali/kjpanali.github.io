Blending Modes

This page provides an overview and explanation of blending modes.

In the "Layers" window in FireAlpaca, there is drop-down menu labeled "Blending." These function in the same way as blending modes in other photo-editing programs like Photoshop and generally share the same names as well. Blending modes are applied to individual layers and determine how the colors of that layer appear and how they interact with the colors of the layers below it. Blending modes are also used for folders, where the blending mode is applied to every layer in the folder as if they were a single layer. This page will focus on the common uses for different blending modes.

By default, the blending mode is set to "Normal." In this setting, colors will appear the same as the selected color and do not affect the appearance of colors in the layers underneath.

Arithmetic Modes

  • After "Normal," the next four blending modes are basic arithmetic operations.

  • "Multiply" multiplies the values of the overlapping colors. This always results in a darker color, so it is often used to apply shadows to an illustration. You may also use this mode to color over scans of digital line art. To do so, set the scanned line art as the bottom layer, and apply "Multiply" blending to all color layers above the line art.

  • Next is "Add" (also known as "Linear Dodge" on other programs), which adds the values of each overlapping color, resulting in a brighter color. Because of that, this mode is often used to add glows and highlights in an illustration.

  • Inversely, "Subtract" subtracts the values of the blending colors from the underlying base colors.

    • Like in math, the order of the layers (which layer is placed higher and thus acts as the blending layer) matters in subtraction. Therefore, "Multiply" and "Add" yield the same color no matter which layer is used for blending, in the same way two numbers are added or multiplied together yield the same result no matter the order in an equation. Meanwhile, using "Subtract" will yield different colors based on which layer the blending mode is applied to. This is also true for "Divide."

  • The "Divide" mode divides the values of the blend layer with those of the base layer and can be used to brighten colors and adjust warmth or coolness. If a layer is blended with a subtract layer of the same color, the result will always be white, in the same way a number divided by itself is always 1.

Lightening and Darkening Modes

  • After the arithmetic modes is "Overlay," which combines the functions of "Multiply" and the next mode, "Screen," which makes colors lighter. As a result, you can use "Overlay" to increase contrast by making light areas brighter and dark areas darker.

  • The next modes are "Lighten" and "Darken," which take only the lightest or darkest values of either layer, respectively. Thus, if "Lighten" is used on an entirely white layer, the result will be completely white because there is no color lighter than white, and "Darken" on a completely black layer will be black because no color is darker than black.

Dodge and Burn Modes

  • "Color Dodge" brightens the colors of the base layer by dividing the base layer by the blending layer. This mode on a white blend layer results in only white, whereas black yields no change. This mode is often used to brighten areas of an image, or to create an illumination effect, such as when illustrating lights. While "Add"can also be used for highlights, "Color Dodge" retains greater contrast.

  • The two burn modes are "Color Burn" and "Linear Burn." "Color Burn" results in a darker color by increasing the contrast between the blend and base layers, while "Linear Burn" decreases brightness. Both modes can be used to create a dark, atmospheric lighting effect, but the blend layer color appears more prominently with "Linear Burn."

Soft and Hard Light Modes

  • "Soft Light" is similar to "Overlay" and can lighten or darken the colors of the base layer, depending on the colors of the blend layer. Black and white do not yield pure black or white; instead, white washes out colors, while black makes colors darker and more saturated. This mode is often used to apply an appearance of diffused light across an illustration.

  • Meanwhile, "Hard Light" creates the appearance of a harsh light, and the blend color is more apparent on the base layer. Using the mode on a completely white blend layer yields pure white, whereas the mode on an all-black layer produces black.

Color Modes

The last four modes are "Hue," "Saturation," "Color," and "Luminosity." These are based on the color dimensions hue, chroma, and luma, with each mode's name indicating the dimension taken from the blend color.

  • "Hue" takes the hue of the blend layer while retaining the luma and chroma of the base layer.

  • "Saturation" takes the chroma of the base layer, or the amount of saturation, and applies only that to the base layer.

  • "Color" takes both the hue and chroma of the top layer, and thus can be used to change the colors of a layers with multiple colors, shadows, or hues.

    • For example, if you have drawn, colored, and shaded the hair of a character and decide to change their hair color, using a color layer allows you to change the colors accordingly, without removing or having to individually recolor shadows or highlights.

  • Lastly, "Luminosity" takes the luma of the top layer while keeping the hue and chroma of the base layer.

Pass Through Mode

Folders have an additional mode known as "Pass Through." Essentially, this determines whether to allow the blending modes of layers within the folder to influence layers outside the folder. For example, if a layer within the folder is set to "Multiply," but the folder itself is set to "Normal," that layer will not be multiplied with the underlying layers, whereas if it the folder is set to "Pass Through," the "Multiply" blending mode will be applied. The mode of a folder also determines whether it can be clipped to underlying layers. For example, if you would like the contents of a folder to only appear on the area marked by an underlying layer, if the folder is set to "Pass Through," marked spaces that lie outside the boundary of the underlying layer will still appear. To make sure the contents of a folder remain within the area of the layer it is clipped to, it must be set to "Normal."
